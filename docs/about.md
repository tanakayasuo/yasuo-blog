# このサイトについて

**やすおのブログ** へようこそ！

## サイトについて

このサイトは、住まいの修繕の仕事をしている「やすお」が、日々の工事の様子や修繕の知識を綴ったブログです。

実際の工事の現場から学んだことを、できるだけ分かりやすくお伝えしています。

## 記事のカテゴリ

- **修繕工事の記録** - 実際に行った修繕作業の詳細な記録
- **修繕のコツ** - 修繕作業で役立つヒントやテクニック
- **ツール・材料** - おすすめの工具や材料の紹介

## サイト運用について

このサイトは **VitePress** という静的サイトジェネレーターで作られています。

Markdown形式で記事を書くだけで、自動的にきれいなブログサイトに変換されます。
